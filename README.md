# Dojo Javascript

Para rodar o projeto você precisa ter instalado o [git](https://git-scm.com/downloads) e o [node](https://nodejs.org/en/download/).


Clonar a aplicação
```
git clone https://github.com/oxedev/dojo-javascript.git
cd dojo-javascript
```


Instalar as dependências 
```
npm install
```

Rodar os testes 
```
npm test
```
